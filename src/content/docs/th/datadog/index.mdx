deploy Datadog agent ด้วย Helm แล้วต่อ metric, trace, และ log ของ ShopMicro เข้า dashboard และ monitor ที่ทำงานเหมือนกันข้าม cloud

บทเรียนของ module นี้กำลังจะมา — ติดตามที่แถบด้านข้างเมื่อมันทยอยลง
