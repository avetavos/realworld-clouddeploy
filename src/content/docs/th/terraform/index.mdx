เรียนรู้ Terraform แบบลงมือ: provider, resource, state, variable และ output ด้วยการสร้าง resource เล็ก ๆ ตัวแรกบนแต่ละ cloud ทั้งสาม

บทเรียนของ module นี้กำลังจะมา — ติดตามที่แถบด้านข้างเมื่อมันทยอยลง
