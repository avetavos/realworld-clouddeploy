สร้าง network module ที่ใช้ซ้ำได้ — VPC บน AWS และ GCP, VNet บน Azure — พร้อม subnet, routing, และ security group ที่ cluster อยู่ข้างใน

บทเรียนของ module นี้กำลังจะมา — ติดตามที่แถบด้านข้างเมื่อมันทยอยลง
