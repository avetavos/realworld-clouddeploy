provision managed Kubernetes บนแต่ละ cloud — EKS, GKE, และ AKS — เป็น module ที่ใช้ซ้ำได้ พร้อม node pool และ kubeconfig ที่ชี้ kubectl ได้

บทเรียนของ module นี้กำลังจะมา — ติดตามที่แถบด้านข้างเมื่อมันทยอยลง
